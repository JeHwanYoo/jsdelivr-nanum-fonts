# Usage

```
@import url(https://cdn.jsdelivr.net/gh/JeHwanYoo/jsdelivr-nanum-fonts@1.0.0/NanumFonts.css);

tag {
  font-family: 'NanumBarunGothic', sans-serif;
}
```

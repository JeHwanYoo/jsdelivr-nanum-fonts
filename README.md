# Usage

```
@import url(https://cdn.jsdelivr.net/gh/JeHwanYoo/jsdelivr-nanum-fonts@1.0.2/NanumFonts.css);

tag {
  font-family: 'NanumBarunGothic', sans-serif;
  font-weight: /* If you want to use NanumBarunGothicBold, use bold (= 700). */
}
```
